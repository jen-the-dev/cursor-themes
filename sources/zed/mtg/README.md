# Add Zed MTG theme JSON files here, then run:

```bash
python3 scripts/zed_to_cursor.py
./install.sh
```

Expected filenames: `mtg-*.json` or any `*.json` in this folder.
They are converted into `themes/mtg/` and registered in `package.json` automatically.

Source repo: https://github.com/jen-the-dev/zed-mtg-themes
